## Use Case 1 – Application Entry & Welcome Message

### Description

This use case represents the **entry point of the Hotel Booking Management System**.

The application starts execution from the `main()` method and displays a welcome message to confirm that the system has started successfully.

At this stage, the program only initializes the system and prints a basic message. No room management, booking logic, or user interaction is implemented yet. The goal is to establish a clear and predictable starting point for the application.

### Program Output

```
Welcome to the Hotel Booking Management System
System initialized successfully.
```

### Concepts Demonstrated

* Java program entry point (`main()` method)
* Basic console output
* Application initialization structure
